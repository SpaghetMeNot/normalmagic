    Tools Overview ¶

    normalMagic is comprised of several tools. Some modify normals, some modify geometry and normals, and some visualize normals and related attributes.

    Custom Normals

    Most of these tools create or edit custom normals. These can be unintuitive to work with if you're unfamiliar Once custom normals exist on a mesh, some default Blender tools/modifiers might not behave as expected. Tools are included to fix situations such as mirroring or beveling but it's something to be aware of.
